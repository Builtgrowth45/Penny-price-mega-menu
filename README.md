# Penny Price Aromatherapy - Mega Menu Preview

This is a client preview of the mega menu design for Penny Price Aromatherapy.

## Features

- Full-width mega menus
- Aromatherapy: Collection image + Therapist CTA
- Health & Wellbeing: 4 featured collections
- Aromatherapy Courses: 4 course images
- About Us: 4 featured sections

## View Demo

Open `index.html` in your browser to see the mega menu in action.

## Navigation Items

1. **Aromatherapy** - Essential oils with therapist discount CTA
2. **Health & Wellbeing** - Wellness products and collections
3. **Aromatherapy Courses** - Educational offerings
4. **About Us** - Company information and resources
